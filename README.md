# Денис Михалыч

Аркадная мини-игра «лови еду ртом» под Telegram Mini App. Один самодостаточный
HTML-файл: рендер на Canvas 2D, синтез звука через Web Audio API, без внешних
зависимостей и без шага сборки.

## Геймплей

- Персонаж внизу экрана, управление свайпом или стрелками влево/вправо.
- Сверху падает еда — ловишь ртом, получаешь очки.
- Лопаты и гири ловить нельзя — урон и сброс комбо.
- Каждый 10-й предмет — бонус «килька в томате».
- Система комбо, погода и ветер, босс-файты, вступительная катсцена.

## Прогрессия

Хранится в `localStorage` (ключи с префиксом `gv_`):

| Ключ          | Что хранит                |
|---------------|---------------------------|
| `gv_hi`       | личный рекорд             |
| `gv_coins`    | монеты для магазина       |
| `gv_ach`      | счётчики для достижений    |
| `gv_unlocked` | разблокированные ачивки    |
| `gv_lb`       | таблица лидеров (локально) |

Достижения (около 30+), магазин апгрейдов: больше жизней, шире хитбокс ловли,
чаще бонусы, множитель монет, длиннее замедление.

## Запуск локально

В браузере вне Telegram объект `Telegram.WebApp` недоступен — интеграция
работает в заглушечном режиме, но геймплей, ачивки и магазин полностью
функциональны.

```sh
python3 -m http.server 8787 --directory .
# открыть http://127.0.0.1:8787/index.html
```

## Известные ограничения

- Таблица лидеров хранится в `localStorage` и поэтому локальна для устройства:
  общего рейтинга между игроками сейчас нет. Нужен бэкенд с записью по
  `user.id` из Telegram.
- Весь прогресс на клиенте и не защищён от правки. До привязки наград или
  оплаты требуется серверная валидация `initData` (HMAC по токену бота).
- Идентификация игрока берётся из `initDataUnsafe` — годится только для
  отображения имени, доверять ей на сервере нельзя.

## Структура

- `index.html` — разметка и оверлеи (splash, game over, лидерборд, достижения, магазин).
- `style.css` — стили.
- `game.js` — вся игровая логика (рендер, спавн, бонусы, боссы, ачивки, лидерборд).
- `worker/` — бэкенд: лидерборд (общий + недельный) и рефералы на Cloudflare Worker.

`game.js` подключён в конце `<body>` — после игровых элементов, на которые
ссылается при инициализации.

## Управление

- Свайп влево/вправо или стрелки — движение.
- Кнопки в оверлее: играть, таблица лидеров, достижения, магазин.
- Тумблеры звука и вибрации в верхней панели.
