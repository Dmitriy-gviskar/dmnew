# Бэкенд лидерборда (Cloudflare Worker)

Общий рейтинг для игры «Денис Михалыч». Хранит топ-100 в KV, валидирует
подпись `initData` от Telegram WebApp (HMAC по токену бота), отдаёт топ-10.

## Эндпоинты

- `POST /score` — тело `{ "initData": "<raw>", "score": <int> }`. Проверяет
  подпись, обновляет личный максимум игрока. Ответ: `{ ok, rank, best }`.
- `GET /top` — топ-10: `{ top: [{n, score}], rank, best, period }`. Параметр
  `?period=week|all` (по умолчанию `all`) выбирает недельный или общий рейтинг.
  С `&initData=<raw>` дополнительно возвращает место и рекорд игрока.

Недельный топ хранится под ключом `top:<ISO-неделя>` (напр. `top:2026-W23`) с
авто-истечением через 60 дней; общий — под ключом `top`. POST /score обновляет
оба.

Идентификаторы игроков наружу не отдаются. `initData` старше 24 часов
отклоняется (защита от повтора).

## Деплой

Нужен аккаунт Cloudflare и токен бота Telegram (от @BotFather).

```sh
cd worker
npm i -g wrangler          # если ещё нет
wrangler login

# создать KV и вставить выданный id в wrangler.toml -> kv_namespaces[0].id
wrangler kv namespace create KV

# положить токен бота секретом (в файлы не коммитим)
wrangler secret put BOT_TOKEN

wrangler deploy
```

После деплоя `wrangler` напечатает URL воркера. Его нужно прописать в
`index.html` в константу `LB_API` (без слэша на конце).

## Замечание про конкурентность

Топ хранится в одном ключе KV и переписывается целиком. При очень высокой
частоте одновременных отправок возможна потеря отдельных обновлений. Для
текущего масштаба игры это приемлемо; при росте — перейти на Durable Object
или D1 с атомарным upsert.
